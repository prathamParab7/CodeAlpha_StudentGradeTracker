# Student Grade Tracker

A simple Java console application to manage and track student grades. 

## Features
- Add, update, and remove students.
- Track grades for different subjects.
- Calculate average grades and display student performance.

## Project Structure
- `Student.java`: Represents a student with their details and grades.
- `GradeManager.java`: Core logic for managing students and calculating statistics.
- `Main.java`: The main entry point and user interface for the application.

## How to Run
1. Make sure you have the Java Development Kit (JDK) installed.
2. Open a terminal or command prompt and navigate to the project directory.
3. Compile the Java files:
   ```bash
   javac *.java
   ```
4. Run the application:
   ```bash
   java Main
   ```
